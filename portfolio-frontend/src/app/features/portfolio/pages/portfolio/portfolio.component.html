The client portal is a Business to Business application, that shows Alpha's client data about their projects. In this project I have taken the role of designer and front-end developer.

The web application has been made using Angular and designed by myself using Figma. The application uses a REST api, that provides data from Alpha's databases. The data is then transformed on the front-end to create the format needed for charts and other widgets.

The project has allowed me to sharpen my greenfield project development and also sharpen my skills in Angular. Solidifing concepts like, storing local user data, state management, authentication, error handling, request handling and creating an expandable/reusable code base.

This project is currently in use by Alpha Clients.

Worx 4.0 is the new version of LTC Worx 3.5, a Business to Business Project managment application. In this project I have taken the role of designer and front-end developer.

Worx 4.0 is a web application that can be used to create project management apps, including a site builder and a code abstraction tool to create logic for the application using the web based UI. The web application has been made using React and designed by myself using Figma.

Developement on Worx as allowed me to work with a large professional team of front-end developers, back-end developers, application developers and QA. In an Agile enviroment.

The Worx 4.0 application is still in-developement.

The website was made to allow Alpha's marketing team to constantly produce new content and be able to put that new content up as easy and quick as possible.

Developed with a large team, including myself in a small wordpress developer group, freelance design, a large team dedicated to specification and ideas and QA.

AlphaCRC.com was completed in the first half of 2020 and is now online for the public to see.
