Worx is a Business to Business Project managment application, providing a unique way of managing workflows in a variety of scenarios. My role in this project has been to produce modern and professional Web based portals for users to interact with Worx.

Producing 2 portals that provide data and actions that user can perform, for Clients and Suppliers. These portals have been built using Angular and an API that was built by another Development team to connect with the Worx application.

Developement on Worx as allowed me to work with a large professional team of front-end developers, back-end developers, application developers and QA.

The Worx portals are still in-developement.

AlphaCRC.com is the company website of Alpha CRC, made using wordpress and elementor. The website shows off the current state of Alpha and provides links to a large amount of resources produced by Alpha.

The website was made to allow Alpha's marketing team to constantly produce new content and be able to put that new content up as easy and quick as possible.

Developed with a large team, including myself in a small wordpress developer group, freelance design, a large team dedicated to specification and ideas and QA.

AlphaCRC.com was completed in the first half of 2020 and is now online for the public to see.

JandJLandscapes.co.uk is a small company website, made using wordpress and elementor. The website was built for my brothers landscape gardening business.

The website was built entirely solo, including acquiring a domain, domain upkeep and hosting and then developing the website.
