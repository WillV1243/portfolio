Design

The design was produced using Material design as a guideline. Angular Material components were used throughout the application to further align with Material Design.

Figma, an online UI prototyping tool, was used to create many iterations of UI layout until the final version was reached.

The colour scheme and logo are customizable through a custom theme system as well as Angular Materials built in theme system.

Dashboard

The dashboard consists of user customizable Widgets. The widgets can be moved and resized using Drag and Drop, created with Angular Gridster. Using ngrx/store to control application state as well ngrx/effects, the dashboard can be customized and saved to the backend.

This allows each user to have a unique dashboard, personal to them.

Create Request

Custom made wizard style form with form progress to allow Clients to create requests.

Wizard function was created using ngrx. Custom form inputs were made following Material design, including a multi-language-select and drag & drop file upload. Custom validation was also created for file upload and many other user inputs throughout the Wizard.

Widgets

Widgets consist of data tables, created using Material Table and customized CDK components.

Each Widget can be customized in many different ways. Including filters for each column, filters for type of data, removing/adding and changing position of columns, widget name and default sorting. These settings are all configured using a dialog then saved to the backend so that users can completely customize their dashboard.

Widgets are also capable of being maximized to fit the full screen and individually reloaded to refresh data.

Localization

The application contains dynamic localization.

The users language is detected automatically through the browser and can be switched at any during use. This was achieved with the library ngx-translate. French, German and english are supported currently but new languages can be added with ease, using externalized JSON files.
